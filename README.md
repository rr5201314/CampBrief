# CampBrief

> 面向大学生的机会信息简报：用更少的时间，找到值得关注的竞赛、考试和资讯。

CampBrief 将分散的信息整理到一个清晰的入口，帮助你快速判断今天有什么值得看、什么需要尽快行动。

## 你可以在这里看到

- **近期竞赛**：浏览值得关注的竞赛与活动，了解基本时间和参与方向。
- **考试信息**：优先查看当前可报名的考试；进入详情可继续查看考试介绍和官方入口。
- **资讯**：首页呈现近三日内容；在每日资讯栏目中可继续浏览学习、AI、技术和体育等信息。

## 怎么使用

1. 从首页开始，快速浏览近期竞赛、可报名考试和近三日资讯。
2. 看到感兴趣的考试或资讯，点击卡片进入详情，获取更多说明和来源链接。
3. 在“每日资讯”中按分类、时间范围或关键词筛选；需要时可选择近 24 小时、3 天、7 天、30 天或指定日期。

## 信息提示

CampBrief 负责整理公开信息，方便你发现机会；涉及报名时间、资格条件、费用和政策时，请始终以对应官方渠道的最新公告为准。

## 持续完善中

我们会持续优化内容整理方式和阅读体验，让重要信息更容易被看见。
