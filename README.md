# CampBrief 🎯

> 大学生AI信息助理 — 每日自动聚合比赛、技术、AI学习信息，智能推荐给每个人

## ✨ 核心功能

- 🏆 **竞赛信息聚合** — 自动收集全网比赛、报名信息
- 💻 **技术趋势追踪** — GitHub Trending 等技术动态
- 🤖 **AI学习资源** — 精选免费课程、实训资源
- 📰 **每日信息差日报** — AI智能筛选，个性化推荐

## 🛠️ 技术栈

- 前端：HTML / CSS / JavaScript
- 数据采集：AI Agent 自动化
- 部署：GitHub Pages

## 📁 项目结构

```
CampBrief/
├── index.html           # 首页
├── competition/         # 比赛板块
├── tech/               # 技术趋势板块
├── ai-learning/        # AI学习板块
└── docs/               # 文档资料
```
